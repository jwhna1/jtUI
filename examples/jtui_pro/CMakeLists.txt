# jtui_pro —— jtUI Pro 子产品 hero example（水青黄金 #2DD4BF，"Built for teams that ship." 调性）
# 维护：曾能混 <jwhna1@gmail.com>

set(JTUI_PRO_SOURCES
    main.cpp
)

if(WIN32)
    add_executable(jtui_pro WIN32 ${JTUI_PRO_SOURCES})
    if(MINGW)
        target_compile_options(jtui_pro PRIVATE -municode)
        target_link_options(jtui_pro PRIVATE -municode)
    endif()
else()
    add_executable(jtui_pro ${JTUI_PRO_SOURCES})
endif()

target_include_directories(jtui_pro PRIVATE
    ${CMAKE_CURRENT_SOURCE_DIR}
    ${CMAKE_CURRENT_SOURCE_DIR}/..)

# ── 编译期嵌入 hero_bg.png ────────────────────────────────────────
# 把 assets/hero_bg.png 字节序列烤进 hero_bg_embedded.h（C 数组）。
# 单二进制部署，运行时不依赖外部文件路径 —— 之前用 JTUI_PRO_ASSETS_DIR 宏
# 注入 Linux 源码路径，mingw 编出来的 exe 在 Windows 上找不到该路径，
# 触发"not found"红字 fallback。嵌入后跨平台行为完全一致。
set(_jtui_pro_png "${CMAKE_CURRENT_SOURCE_DIR}/assets/hero_bg.png")
set(_jtui_pro_embedded "${CMAKE_CURRENT_BINARY_DIR}/hero_bg_embedded.h")
file(READ "${_jtui_pro_png}" _jtui_pro_png_hex HEX)
# 每两个 hex 字符前加 "0x"，每 32 字节换行（避免单行 1.6MB 让某些编辑器卡）
string(REGEX REPLACE "(................................)" "\\1\n    "
       _jtui_pro_png_hex "${_jtui_pro_png_hex}")
string(REGEX REPLACE "([0-9a-f][0-9a-f])" "0x\\1," _jtui_pro_png_bytes
       "${_jtui_pro_png_hex}")
file(WRITE "${_jtui_pro_embedded}"
"// Auto-generated by CMake from assets/hero_bg.png — DO NOT EDIT.\n"
"#pragma once\n"
"#include <cstddef>\n"
"#include <cstdint>\n"
"namespace jtui_pro {\n"
"inline constexpr unsigned char kHeroBgPng[] = {\n    "
"${_jtui_pro_png_bytes}\n"
"};\n"
"inline constexpr std::size_t kHeroBgPngSize = sizeof(kHeroBgPng);\n"
"}\n")
# 让 main.cpp 能 #include 生成的头
target_include_directories(jtui_pro PRIVATE ${CMAKE_CURRENT_BINARY_DIR})

target_link_libraries(
    jtui_pro
    PRIVATE
        jtui::api
)

hui_configure_target(jtui_pro)
